## セットアップ & 実行方法

詳細なセットアップと実行方法は、`frontend` と `backend` の各ディレクトリ内にある `README.md` を参照

### 1. バックエンド (FastAPI)

1.  プロジェクトのルートディレクトリ (`task-management-app/`) で、Python 3 を使い仮想環境 (`.venv`) を作成します。

## 主な機能

### 必須要件
* タスクの一覧表示
* タスクの新規作成
* タスクの削除
* タスクの完了/未完了の切り替え

### 実装済みの加点要素
* **TypeScriptの使用**
* **エラーハンドリング** 
* **バリデーション** 
* **テストコードの追加** 
* **レスポンシブデザイン**
